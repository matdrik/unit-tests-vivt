# unit-tests-vivt

## Технологии

TypeScript + Node.js + Jest

## Запуск

1. Клонировать/скачать проект

    ```
    git clone https://github.com/matdrik/unit-tests-vivt && cd unit-tests-vivt
    ```

2. Установить зависимости (в папке проекта выполнить)

    ```
    npm install
    ```

3. Запуск тестов

    ```
    npm test
    ```
